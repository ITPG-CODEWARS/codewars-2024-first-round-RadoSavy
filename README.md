За този код ми помогна най-моного господина ми по Англиски език с идея как да усъществия този код и класния ми ръководител ползвайки найните връзки да ми осигури човек, който работи в БДЖ за да ми акцентира върху някой влакове и да пи ги покаже. При създаването на този код съм използвал множество front-end библиотеки, като StackPath, Cloudflare, Google Fonts, jQuery и други. За правилното стартиране на проекта, бих препоръчал използването на VS Code, тъй като предлага най-лесния начин за работа с него.
